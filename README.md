# Maia Chess Bot

Chess bot that acts simply on pattern recognition rather than brute forcing every possible move. Trained on linux w/ amdgpu. Essentially a "drunk" human GM. (Bot already exists, but this is just inspired and trained from scratch to understand the process better)
